# Portfolio

Personal portfolio site showcasing projects, built with React and Tailwind CSS. Deployed on Vercel.

## Tech Stack

- React
- React Router
- Tailwind CSS
- Lucide React

## Local Development

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Project Data

Projects are defined in `src/projects.js`. Update `PROJECT_ORDER` in that file to change display order.
